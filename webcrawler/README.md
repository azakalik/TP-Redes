# Scraping
## About
This scrapes mercadolibre.com.ar for different car brands.

## Requirements
- Python 3.10
- Pip3
- Pipenv (install with `pip3 install pipenv`)
- Docker

## Initialization
1. `cd webcrawler`
2. `pipenv install`

## Creating (and recreating) the image
`pipenv run build`

## Running the image
`pipenv run scrape`
